# Spike Skill

Disposable feasibility experiments before planning or implementation.

Use `spike` when the useful next move is learning, not production code. Spike notes live in `.agent/evidence/<run-slug>/spike/SPIKE_NOTES.md` and feed back into `plan`, `counsel`, or `repo-memory`.

## Installation

```bash
ln -sfn ~/Developer/Personal/skill-spike/skills/spike ~/.claude/skills/spike
ln -sfn ~/Developer/Personal/skill-spike/skills/spike ~/.codex/skills/spike
```
